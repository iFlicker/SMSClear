# SMSClear
本工具可以根据指定黑名单和白名单以及限定时间范围筛选本地短信，然后删除。

V1.1 支持结果页多选(默认多选)

V1.0 基础功能

注意：  
  1.一旦删除无法恢复请谨慎操作  
  2.部分手机无法通过弹框临时设置应用为系统默认短信应用，需要跳转设置才能有效删除，删除完毕必须设置回系统自带的默认短信应用，否则短信功能将无法正常运行
